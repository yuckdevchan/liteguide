# Proprietary Software
[Proprietary software](proprietary-software) is **software** that **does not permit** users to **freely use**, **modify**, or **distribute it**.

It is **likely** that **most software** you use is **proprietary software**, though you **almost definitely** use some [open-source](open-source-and-copyleft) software **as well**.

(w) **Proprietary software** is also known as **closed-source** software, **commercial** software, or **non-free** software (**mainly** by **supporters** of the [Free Software Movement](free-software-movement)).

(w) **Software** which has **viewable source code** but does not **allow modification** or **redistribution** is still **considered proprietary software**.

(i) Publishing **proprietary software** is like giving someone **orange juice** in a **sealed container**. They can **drink it**, but they **cannot see** how it was **made**, **change the recipe**, or **share it with others**.
