# DOS
[DOS](https://en.wikipedia.org/wiki/DOS) stands for Disk Operating System.

The source code for 86-DOS 1.00 was published in 2026.

## MS-DOS
[MS-DOS](https://en.wikipedia.org/wiki/MS-DOS) (Microsoft Disk Operating System) is a result of Microsoft's purchase of 86-DOS in 1981. It was the foundation for early versions of [Windows](http://localhost:5174/guides/software/operating-systems/other/microsoft-windows).

The source code for MS-DOS [v1.1 and v2.0](https://github.com/michaelmsonne/MS-DOS) were published in 2018, [v1.25](https://github.com/microsoft/MS-DOS/tree/main) and v2.11 in 2018, and [v4.00 in 2024](https://github.com/microsoft/MS-DOS/tree/main).
