# Microsoft Windows
[Microsoft Windows](https://en.wikipedia.org/wiki/Microsoft_Windows) is a series of operating systems developed by Microsoft. From 1985 to 2001, the home user versions were built on top of MS-[DOS](/guides/software/operating-systems/other/dos), a separate operating system.

Since Windows XP in 2001, home user versions have been built on top of the newer and specially-developed NT [kernel](/guides/software/operating-systems/kernels).
