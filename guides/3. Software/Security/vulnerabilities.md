# Vulnerabilities
A [software vulnerability](https://en.wikipedia.org/wiki/Vulnerability_(computer_security)) is a type of bug in computer software which can be exploited to perform a malicious action.

## Common Causes
- Memory errors. Languages such as [Rust](rust) seek to prevent memory errors from being possible.
- Input validation errors. A user may be able to trick a program into executing their input as instructions. (See [SQL injection](sql-injection) and [cross-site scripting](cross-site-scripting))
